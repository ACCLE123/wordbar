# WordBar

一个简洁的 **Mac 状态栏背单词工具**。  
目标用户：程序员 & 学习雅思的同学，可以在工作/摸鱼间隙利用碎片化时间记单词。  

## ✨ 功能目标
- 状态栏直接显示 **单词 | 中文释义**  
- 定时切换单词（如每 3 分钟）  
- 右键菜单支持：下一词 / 退出  
- 支持雅思核心词库、间隔重复记忆  

## 📌 当前进度
- ✅ SwiftUI + AppKit 集成，状态栏成功显示单词  
- ✅ 右键菜单：切换单词 & 退出  
- ⏳ 待实现：定时自动切换单词  
- ⏳ 待实现：词库导入 & 复习逻辑  

## 🛠 技术栈
- **语言**：Swift 5 + SwiftUI  
- **平台**：macOS（状态栏常驻应用）  
- **UI**：NSStatusBar + NSMenu
